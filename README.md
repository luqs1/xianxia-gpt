# XianXia-GPT

A tiny gpt model fine-tuned to generate Chinese web novel-style content with English characters.

## Overview

XianXia-GPT is a lightweight language model inspired by NanoGPT, specifically trained to generate content in the style of Chinese cultivation novels (Xianxia), wuxia, and other popular Chinese web novel genres - all in English text.

## Features

- Generates cultivation-themed narratives with appropriate terminology
- Understands common tropes in Chinese web novels
- Creates compelling characters and story arcs in the Xianxia style
- Lightweight implementation based on NanoGPT architecture

## Training

The model was trained on a curated dataset of English translations of popular Xianxia, Xuanhuan, and Wuxia novels. If you have access to epubs or pdfs. I encourage you to use the calibre CLI or other converters to get novels in a raw text format first.
